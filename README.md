# 🏋️ Fitness Buddy AI

An AI-powered virtual fitness coach built using **IBM watsonx Orchestrate** and **IBM Granite Foundation Models**. Fitness Buddy provides personalized workout plans, healthy meal suggestions, motivation, and wellness guidance through natural conversations.

---

## 📌 Problem Statement

Maintaining a healthy lifestyle is difficult due to busy schedules, lack of personalized guidance, inconsistent motivation, and limited access to professional fitness coaching. Fitness Buddy addresses these challenges by providing intelligent, personalized fitness assistance anytime and anywhere.

---

## 🚀 Features

- 💪 Personalized workout recommendations
- 🥗 Healthy meal and nutrition suggestions
- 🌟 Daily motivation and habit-building tips
- 📚 Fitness knowledge and wellness guidance
- 🧠 AI-powered conversational assistant
- 🔒 Safe and responsible fitness recommendations
